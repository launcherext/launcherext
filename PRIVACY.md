# Privacy Policy for Launch Ext

**Last Updated:** January 12, 2026

## Introduction

Launch Ext ("we", "our", or "the extension") is a Chrome extension that helps users generate AI-powered banners and launch tokens on pump.fun. We are committed to protecting your privacy and being transparent about our data practices.

## Data We Collect

### 1. Wallet Information
- **What:** Public wallet addresses when you connect your wallet
- **Why:** To track your token launches and display your portfolio
- **Storage:** Stored locally in your browser and on our servers
- **Sharing:** Never shared with third parties

### 2. Launch Data
- **What:** Token names, symbols, signatures, and mint addresses of tokens you create
- **Why:** To display your launch history and calculate statistics
- **Storage:** Stored locally and on our backend database
- **Sharing:** Public blockchain data only

### 3. Generated Images
- **What:** AI-generated banner images you create
- **Why:** To display in your launch history
- **Storage:** Temporarily stored during generation, then on IPFS (decentralized)
- **Sharing:** Public via IPFS as part of token metadata

### 4. Usage Analytics
- **What:** Extension usage patterns (anonymous)
- **Why:** To improve the extension and fix bugs
- **Storage:** Aggregated data only, no personal identifiers
- **Sharing:** Never shared

## Data We DO NOT Collect

- ❌ Private keys (never leave your device)
- ❌ Passwords
- ❌ Transaction details beyond public blockchain data
- ❌ Personal information (name, email, etc.)
- ❌ Browsing history
- ❌ Data from other websites or extensions

## How We Use Your Data

1. **Provide Services:** Display your launch history, calculate P/L, show token stats
2. **Improve Experience:** Optimize features based on usage patterns
3. **Debugging:** Identify and fix bugs
4. **Security:** Detect and prevent abuse

## Data Storage & Security

### Local Storage
- Encrypted wallet keys (if using embedded wallet)
- Launch history
- User settings

### Backend Servers
- Launch data (public blockchain info)
- Usage statistics (anonymized)
- Hosted on secure cloud infrastructure (Vercel/Railway)

### Security Measures
- **Encryption:** All private keys encrypted at rest
- **HTTPS:** All API communications use HTTPS
- **Rate Limiting:** Prevents abuse and spam
- **Access Control:** Strict backend access policies

## Third-Party Services

We use the following third-party services:

1. **PumpPortal API**
   - Purpose: Token creation on pump.fun
   - Data Shared: Token metadata, mint addresses
   - Privacy Policy: https://pumpportal.fun/privacy

2. **IPFS (via pump.fun)**
   - Purpose: Store token images and metadata
   - Data Shared: Public token information
   - Note: IPFS is decentralized and permanent

3. **Solana RPC Providers**
   - Purpose: Blockchain interactions
   - Data Shared: Public blockchain transactions
   - Privacy Policy: Varies by provider

4. **AI Image Generation Service**
   - Purpose: Generate token banners
   - Data Shared: Text prompts only
   - Privacy Policy: [Your AI provider's policy]
